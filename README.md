## 🔋 Getting Started

Para levantar el proyecto, correr los siguientes comandos:

1- Instalar dependencias:
```bash
npm i
```
2- Setear variables de entorno:
```bash
cp .env.template .env
```
2- Levantar el proyecto:
```bash
npm run dev
```
